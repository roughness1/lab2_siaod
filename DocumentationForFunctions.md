# BrowserHistory - Документация

Данный документ описывает классы и методы простого трекера истории браузера на Python.

## Общая информация

Модуль содержит два класса:

BrowserHistoryRecord - представляет одну запись в истории
BrowserHistory - управляет списком посещений, навигацией и аналитикой

## Класс BrowserHistoryRecord

### Описание

Представляет одну запись в истории браузера

### Атрибуты

url (str) - посещённый URL

time - время посещения

flag - дополнительный флаг или метка

### Конструктор

BrowserHistoryRecord(url, time, flag)

#### Параметры:
url - адрес страницы

time - время посещения

flag - метка записи

## Класс BrowserHistory

### Описание

Реализует хранение истории браузера с навигацией назад и вперёд, поиском, аналитикой и сохранением в файл

### Конструктор

BrowserHistory()

Инициализирует пустую историю и устанавливает текущий индекс -1

## Методы

### add_record(url, time, flag)

Добавляет новую запись в историю. Если пользователь находится не в конце истории, будущие записи удаляются

#### Параметры:
url - URL страницы

time - время посещения

flag - дополнительная метка

### clear_history()

Очищает историю и сбрасывает индекс на -1

### print_all()

Выводит все записи истории в консоль

#### Формат вывода:
url time flag

### forward()

Переходит вперёд по истории если возможно

Возвращает запись или None если движение невозможно

### back()

Переходит назад по истории если возможно

Возвращает запись или None если движение невозможно

### _get_domain(url)

Внутренний метод извлекает домен из URL

Параметры:
url - строка URL

Возвращает домен или путь если домен отсутствует

### top_n_transitions(n)

Находит самые частые переходы между доменами

#### Параметры:
n - количество результатов

Возвращает список переходов и их частоту

### save_to_file(filename)

Сохраняет историю в файл с Base64 кодированием

#### Параметры:
filename - имя файла

Формат строки:
base64("url|time|flag")

### search_by_domain(domain)

Ищет записи по домену

#### Параметры:
domain - домен

Возвращает список записей
